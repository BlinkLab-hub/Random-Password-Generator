# Random Password Generator

A simple web-based password generator built with **HTML**, **CSS**, and **JavaScript**. This application allows users to generate random passwords with customizable character sets and password lengths.

## Features

* Generate passwords of a custom length
* Include:

  * Numbers (`0-9`)
  * Uppercase letters (`A-Z`)
  * Lowercase letters (`a-z`)
  * Symbols (`!@#$%^&*()_-+=/`)
* Input validation for minimum password length
* Error messages for invalid input
* Responsive and modern user interface
* Floating label animation for the input field

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* SVG icons

## How It Works

1. Enter the desired password length.
2. Select one or more character sets:

   * Numbers
   * Uppercase letters
   * Lowercase letters
   * Symbols
3. Click **Generate Password**.
4. The generated password will be displayed at the bottom of the card.

## Validation Rules

* Password length must be greater than 3.
* At least one character set must be selected.
